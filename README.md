## 🇹🇷 Türkçe

Tekla model dosyalarını hızlıca analiz eden Windows masaüstü uygulaması.

### Öne Çıkanlar
- Modern ve sade arayüz
- Sürükle-bırak desteği
- Tarama geçmişi görüntüleme
- Geçmiş temizleme
- Türkçe / İngilizce dil desteği

### Kullanım
1. Releases bölümünden Windows EXE dosyasını indirin.
2. Uygulamayı çalıştırın.
3. Model dosyanızı seçin veya sürükleyip bırakın.
4. Sonucu ekranda görüntüleyin.

👉 Daha fazla araç ve güncellemeler için: **[zadax.net](https://www.zadax.net)**

## 🇬🇧 English

A Windows desktop tool to quickly analyze Tekla model files.

### Highlights
- Clean modern interface
- Drag & drop support
- Scan history view
- Clear history action
- Turkish / English language support

### Usage
1. Download the Windows EXE from Releases.
2. Launch the app.
3. Select your model file or drag & drop it.
4. View the result on screen.
